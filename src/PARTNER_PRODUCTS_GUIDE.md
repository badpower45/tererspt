# دليل إدارة منتجات الشراكات - AddValues ERP

## نظرة عامة

نظام متقدم لإدارة ومقارنة منتجات الشركاء مع ميزات تحليل شاملة لاتخاذ قرارات شراء ذكية.

---

## الميزات الرئيسية

### ✅ 1. كتالوج شامل للمنتجات
- عرض جميع منتجات الشركاء في مكان واحد
- تصنيف حسب الفئة
- بحث متقدم
- فلترة ديناميكية

### ✅ 2. وضع المقارنة (Comparison Mode)
- مقارنة منتجات متشابهة من شركاء مختلفين
- عرض جنباً إلى جنب في جدول
- تمييز أفضل الأسعار
- مقارنة المواصفات الفنية

### ✅ 3. معلومات تفصيلية
- الأسعار والخصومات
- التوفر (متوفر، تحت الطلب، غير متوفر)
- مدة التوصيل
- فترة الضمان
- التقييمات

### ✅ 4. خصومات الجملة
- عرض مستويات الخصم
- حساب تلقائي للسعر النهائي
- تشجيع الطلبات الكبيرة

## واجهة المستخدم

### العرض العادي (Grid View)

**مميزات:**
- بطاقات (Cards) كبيرة لكل منتج
- عرض مرئي جذاب
- سهولة التصفح
- معلومات شاملة في كل بطاقة

**محتوى البطاقة:**
- اسم المنتج + رقم SKU
- اسم الشريك
- التقييم (نجوم)
- الفئة (Badge)
- السعر (مميز بلون أزرق)
- خصومات الجملة
- الحالة (متوفر/تحت الطلب/غير متوفر)
- مدة التوصيل
- فترة الضمان
- المواصفات الفنية
- زر "إضافة إلى عرض السعر"

### وضع المقارنة (Comparison Mode)

**مميزات:**
- عرض جدولي
- مقارنة مباشرة
- ترتيب حسب السعر (من الأرخص للأغلى)
- تمييز أفضل سعر بلون أخضر
- عرض جميع التفاصيل جنباً إلى جنب

**أعمدة الجدول:**
1. الشريك
2. المنتج (الاسم + SKU)
3. السعر + Badge أفضل سعر
4. الحالة (مع أيقونة)
5. مدة التوصيل
6. فترة الضمان
7. التقييم
8. المواصفات الفنية

## هيكل البيانات

### PartnerProductCatalog
```typescript
{
  id: string;
  partner_id: string;
  name: string;              // اسم المنتج
  category: string;          // الفئة مثل "Inverter 5KW"
  sku: string;               // رقم المنتج
  price: number;             // السعر
  bulk_discount?: [          // خصومات الجملة
    {
      min_quantity: number;
      discount_percentage: number;
    }
  ];
  specifications: {          // المواصفات الفنية
    power?: string;
    efficiency?: string;
    voltage?: string;
    warranty?: string;
    // ... المزيد
  };
  images?: string[];
  availability: 'in_stock' | 'on_order' | 'out_of_stock';
  delivery_time_days: number;
  warranty_months: number;
  rating?: number;           // من 1 إلى 5
  created_at: string;
  updated_at: string;
}
```

## أفضل الممارسات

### للمشتري
1. **قارن دائماً**: استخدم وضع المقارنة قبل الشراء
2. **راجع التقييمات**: لا تعتمد على السعر فقط
3. **تحقق من التوفر**: اتصل بالشريك قبل الطلبات الكبيرة
4. **استفد من خصومات الجملة**: اطلب كميات أكبر لتوفير المال
5. **احتفظ بسجلات**: وثّق جميع الاتفاقيات

### لمدير الشراكات
1. **حدّث البيانات باستمرار**: أسعار ومواصفات دقيقة
2. **تواصل مع الشركاء**: علاقات قوية = أسعار أفضل
3. **راقب السوق**: كن على اطلاع بالأسعار التنافسية
4. **تفاوض**: لا تقبل السعر الأول
5. **نوّع الشركاء**: لا تعتمد على شريك واحد
